面向对象概念

OOP=Object+Classification+Inheritance+CommunicationOOP=Object+Classification+Inheritance+CommunicationOOP=Object+Classification+Inheritance+Communication

Object​

对象是基本的运行实体

Message​

对象之间通信的一种构造

Classification​

类定义一组大体上相似的对象，类使对象的模板，对象是类的实例(Instance)

Inheritance​

继承是父子类共享数据和方法的机制

Polymorphism​

不同对象收到同一消息产生不同结果

通用多台​

  • 参数多态
  • 包含多态

特定多态​

  • 过载多态
  • 包强制多态

Dynamic Binding​

根据接收对象的具体情况将请求的操作与实现的方法进行连接
