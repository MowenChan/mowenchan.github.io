低级语言和高级语言​

  • 指令是特定计算机系统所固有的
  • 汇编用容易记的符号代替机器指令
  • 高级语言与人类使用的自然语言比较接近

编译程序和解释程序​

  • 汇编
    • 汇编-->汇编程序-->在机器上执行
  • 解释
    • 解释器直接执行源程序
    • 把源程序翻译成中间代码执行
  • 编译
    • 源程序-->目标语言程序-->在机器上执行

程序设计语言定义​

  • 语法
    • 由语言的基本符号组成词序中各个语法成份的一组规则
  • 语义
    • 按语法规则构成语句的含义
  • 语用
    • 各记号和使用者的关系
  • 语境
    • 编译环境
    • 运行环境
