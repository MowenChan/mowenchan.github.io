数组​

  • 定义
    • 数组是定长线性表在维数上的扩展
    • A[b1,b2,⋯ ,bn]A[b_1,b_2,\cdots,b_n]A[b1​,b2​,⋯,bn​]
    • 每一维的下界为 1
    • bib_ibi​是第iii维的上界
    • nnn维数组是一种同构的数据结构
      • 每个元素类型相同
      • 每个元素结构一致
  • 特性
    • 元素数量固定
    • 元素下标有序且受上下界约束
  • 操作
    • 根据下标取元素值
    • 根据下标改元素值

矩阵​

  • 定义
    • 矩阵可理解为对数组的扩展，主要讨论在节省存储空间的情况下使矩阵高效运算
  • 特殊矩阵
    • 矩阵中的元素分布有一定规律
  • 稀疏矩阵
    • 非 0 元素分布没有规律且非 0 元素的个数远远小于 0 元素的个数

广义表​

  • 定义
    • 广义表是线性表的推广
    • LS=(a,(b,c,d))LS=(a,(b,c,d))LS=(a,(b,c,d))
  • 特性
    • 广义表的元素可以是单元素
    • 广义表的元素可以是有结构的表
    • 广义表的元素可以是其他广义表的引用
    • 广义表的元素可以是其递归的自身引用
  • 存储结构
    • 一般使用链式线性表

非线性结构​

树​

  • 定义
    • T(M1,M2,⋯ ,Mn)T(M_1,M_2,\cdots,M_n)T(M1​,M2​,⋯,Mn​)
    • 树是nnn个节点的有限集合
    • 有且仅有一个根节点
    • 其余节点可分为若干个互不相交的有限子集
    • 任何节点可以有 ≥2\ge 2≥2个子结点
  • 特性
    • 一个节点的子树节点的个数称为该节点的度degree
    • 除根节点外度大于 0 的节点称为内部节点inode
    • 度为 0 的节点称为叶子节点leaf
    • 节点的子树称为节点的儿子child
    • 有子树的节点称为子树节点的双亲parent
    • 具有相同双亲的节点互为兄弟sibling
    • 根为第1 层level
    • 一棵树的最大层数称为树的高度/深度depth
    • 树节点从左到右有次序的是有序树
    • 树节点从左到右无次序的是无序树
  • 存储结构
    • 双亲表示法: data∣→parent\boxed{data|\to parent}data∣→parent​
    • 孩子表示法: data∣→child1∣→childk\boxed{data|\to child_1|\to child_k}data∣→child1​∣→childk​​
    • 双亲孩子表示: data∣→parent∣→child1∣→childk\boxed{data|\to parent|\to child_1|\to child_k}data∣→parent∣→child1​∣→childk​​
    • 孩子兄弟/二叉链表表示法: data∣→Lchild∣→Rsibling\boxed{data|\to Lchild|\to Rsibling}data∣→Lchild∣→Rsibling​
  • 遍历
    • 先根遍历
    • 后根遍历
