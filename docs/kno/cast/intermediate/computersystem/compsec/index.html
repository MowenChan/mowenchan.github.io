安全威胁​

  • 授权侵犯
  • 拒绝服务
  • 窃听
  • 信息泄露
  • 截获/修改
  • 假冒
  • 否认
  • 非法使用
  • 人员疏忽
  • 完整性破坏
  • 媒体清理
  • 物理入侵
  • 资源耗尽

影响数卷全的因素​

  • 内部因素
    • 采用多种加密技术
    • 制定数卷全规划
    • 容量、容错、数据保护、数据备份
    • 事故应急容灾措施
    • 重视安全管理、制定数卷全管理规范
  • 外部因素
    • 数据密级，访问权限
    • 身份、密码、口令、指纹、声纹、笔迹认证
    • 防火墙
    • 入侵检测、审计、追踪
    • 防辐射、防水、防火
