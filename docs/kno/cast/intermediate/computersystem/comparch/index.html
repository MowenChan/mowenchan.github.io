RISC​

简化指令总数和指令功能，降低硬件复杂度。

关键技术​

  • Oerlapping Register Windows
    • 设置一个寄存器堆，划分为3个连续窗口和一个公用窗口。
      1. 和前一个过程共用
      2. 和后一个过程共用
      3. 临时窗口
      4. 公用窗口
  • 优化编译技术
    • 通过编译技术的优化来合理分配寄存器
      1. 优化高频指令
      2. 减小高级语言和机器语言的语义差异
      3. 减小操作系统和系统结构语义差异
  • 超流水线，超标量，超长指令字技术
    • SPL细化流水，增加级数，提高主频，用时间换空间
    • SS内装多条流水线，用空间换时间
    • VLIW充分发挥软件作用，从而简化硬件，提高性能，但需要较高的时钟频率
  • 硬布线和微程序
    • SPL
    • SS
    • VLIW

指令控制方式​

  • 顺序
    • 串行执行
    • 控制简单
    • 速度慢，硬件资源使用效率低
  • 重叠
    • 在完成解释第K条指令前就可以开始解释第K+1条指令
    • 速度提高，控制简单
    • 出现冲突，转移，相关问题
  • 流水
